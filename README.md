## task18
```
решена посредством:
1. генерации в создаваемый список случайных числе (в заданном разработчиком диапазоне), размерности, устанавливаемой пользователем
2. нахождением абсолютной разности между первичным элементом и искомым значением
3. сравнением, с помощью цикла for, абсолютной разности искомого от первичного ключа с абсолютной разностью последующих элементов списка с искомым. И, в случае, когда разность приближается к нулю - выводит данный элемент в терминал, как результат работы.
```
